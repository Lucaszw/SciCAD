## Installing From NPM

```sh
> npm install microdrop-3.0
> microdrop-3.0.cmd
```

## Installing From Source

```sh
> git clone https://github.com/Lucaszw/microdrop-3.0
> cd microdrop-3.0
> node index.js
```
